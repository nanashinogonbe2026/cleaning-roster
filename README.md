# 掃除当番表マネージャー V3 (Cleaning Roster V3)

学校や組織における掃除当番、日直、スピーチ等の役割を自動的に、かつ公平に割り振るWebアプリケーションです。
職業訓練校の先生が欠席者などで毎日変わる当番決めに苦労していたので「この面倒くさいルーチンは自動化出来るはず」と思って作りました。
AIを使って何かしたい！少し前の自分には出来なかった、今ならAIの力を使って出来るって事証明出来ました。

## 概要

「Stream & Debt（ストリーム＆負債）」モデルを採用し、欠席や休日によるローテーションのズレを最小限に抑えつつ、公平な役割分担を実現します。欠席者は「負債」として記録され、次回優先的に当番が割り当てられます。

## 主な機能

- **自動ローテーション**: 日直、スピーチ、コメント、掃除当番（ペア）を自動割り当て。
- **負債 (Debt) システム**: 欠席した人は自動的に記録され、後日優先的に割り当て。公平性を担保。
- **スライド式掃除当番**: 掃除当番は毎日ペアが入れ替わる（例：A&B → B&C → C&D）スライド方式を採用。
- **柔軟な設定**:
    - **休日設定**: カレンダーで特定の日を休日に設定可能。
    - **掃除なし設定**: 行事等で掃除だけ無い日も設定可能。
    - **個別欠席**: 急な欠席もワンクリックで登録・スキップ可能。
- **データ管理**:
    - ブラウザに自動保存。
    - JSONファイルへのエクスポート/インポート機能で、データのバックアップや引き継ぎが可能。

## 使用技術

- HTML5
- CSS3 (Tailwind CSS - CDN)
- JavaScript (Vanilla JS)

## 使い方

1. `index.html` をブラウザで開きます。
2. 初回のみ、メンバー人数を入力してセットアップします。
3. 毎日の当番表が自動生成されます。
4. メンバー名の変更や細かい設定は「👥 メンバー・設定」タブから行えます。
